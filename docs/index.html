Get Started in 4 Steps

From zero to full M365 access in minutes.

01

Register Azure App

Create an app in Azure Portal → Entra ID → App registrations. Enable "Allow public client flows".

02

Grant API Permissions

Add delegated permissions for Mail, Calendar, Teams, Files, Sites, and User. Grant admin consent.

03

Install & Build

Clone, install, build.

npm install && npm run build
04

Connect to Claude

Add to Claude Desktop or Claude Code with your Client ID.

claude mcp add msgraph
