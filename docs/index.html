4 ステップで始める

ゼロから Microsoft 365 フルアクセスまで、数分で完了。

1

Azure アプリ登録

Azure Portal → Entra ID → アプリの登録。「パブリック クライアント フローを許可」を有効化。

2

API アクセス許可の付与

Mail, Calendar, Teams, Files, Sites, User の委任アクセス許可を追加。管理者の同意を付与。

3

インストールとビルド

リポジトリをクローンしてビルド。

npm install && npm run build
4

Claude に接続

Claude Desktop / Claude Code にクライアント ID を設定して追加。

claude mcp add msgraph
